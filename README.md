A Web project management application by using Java.

Tools used in the project:

 - Spring Framework 5
 - Spring Boot 2
 - Maven
 - Spring Data CRUD repositories
 - Tymeleaf
 - Hibernate
 - PostgreSQL database
 - Integration Testing
 - Spring AOP
 - Spring security
 - RESTFul Web Services API
 - Bootstrap
